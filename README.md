# Aegis-Link | Strategic Battlespace Awareness System

## 🛡️ Project Overview
Aegis-Link is a Cybersecurity and Signal Intelligence (SIGINT) dashboard designed to detect, classify, and audit RF signals in contemporary conflict zones. It focuses on GNSS integrity and asset identification.

## 🚀 Core Features
- **Asset Identification:** Real-time tracking of brand, model, and velocity for motorcycles and ground vehicles.
- **Electronic Warfare (EW) Detection:** Automated detection of GPS spoofing and signal interference.
- **Forensic Audit:** Polar satellite radar and multi-constellation consensus verification.
- **Strategic Targeting:** Global coordinate resolution using Geopy integration.

## 🛠️ Installation & Setup
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/Aegis_Link.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the dashboard: `streamlit run main.py`
